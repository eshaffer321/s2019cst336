There are three types of loops: for loop, while loop, and do while loop.


for loop:for ( init; condition; increment ) { statement(s); }
The init would be the starting condition. It is the first thing to be executed and only one time.
The next would be the condition. If the condition is true, the for loop continues unless it is false.
Lastly, the increment can either increase or decrease. It can increase or decrease depending on how you want it to increase/decrease. In c++, the ++ is increasing by one and the -- decrements by one.



while loop: while(condition){}
First, it will check if the condition is true. If the condition is true, then it will go inside the while looop.



do while loop: do{statement(s);}while(condition);
The only difference with this loop is it runs at least once. It does the statement first and then it checks the condition if it is true or not.
