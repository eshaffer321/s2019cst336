Vectors: They are similar to arrays where it stores a series of elements using the same varible name. The only difference would be that vectors are dynamically sized which is an advantage. The size does not need to be initialized and it can grow or shrink.

Vectors contain some functions already like:

  1. nameofArray.push_back(val)
  2. nameofArray.pop_back()
  3. nameofArray.insert(first, last)
  4. nameofArray.erase()
  5. nameofArray.swap(val1, val2)
  6. nameofArray.clear()
