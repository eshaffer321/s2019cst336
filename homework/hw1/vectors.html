    Arrays: A collection of elements of the same data type. The size is constant meaning if the size is a certain number, then it cannot be changed unless you locate more memory. If the array size is 5, then it will allocate 5 blocks of memory right next to each other. Arrays are useful because if you need multiple variables for the same data type, then an array can hold that data.
